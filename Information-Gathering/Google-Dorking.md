#  Google Dorking
Google dorking is a technique used by OSINT (Open-Source Intelligence) using advanced search techniques to gather intelligence from publicly available sources indexed by Google. 
It involves using advanced search operators to narrow down search results to specific criteria, such as file types, website domains, or specific keywords.

Google dorks can be crafted to target various types of information, such as:
- Vulnerable websites
- Exposed data
- Network devices
- Specific file types

##  Google Dorks Operators
In Google dorking, an "operator" refers to specific search syntax or commands used to refine search queries and retrieve more targeted results.
These operators allow you to perform advanced searches and access information that may not be readily available through standard searches.

Search for specific keywords within the URL of indexed web pages
- `inurl:"register"`
